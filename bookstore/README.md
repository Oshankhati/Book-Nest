Bookstore Management System REST API

Technologies Used
- Java
- Spring Boot
- Spring Security
- JWT Authentication
- MySQL
- Swagger

How to Run

1. Install MySQL
2. Create database:

CREATE DATABASE bookstore_db;

3. Update application.properties with your MySQL username/password

4. Run project

mvn spring-boot:run

API Documentation

http://localhost:8081/swagger-ui/index.html
